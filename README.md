# Results_paper_SHEAP

This repository contains the results obtained with **Spectral Handling and Estimation of AGN Parameters** (**SHEAP**), the first GPU-based code designed for AGN spectral fitting.

## Installation

Create and activate a Conda environment with Python 3.12:

```bash
conda create -n papersheap python=3.12
conda activate papersheap
````

Go to the directory where the repository is located:

```bash
cd path/to/Results_paper_SHEAP
```

Install the requirements:

```bash
pip install -r requirements.txt
```
